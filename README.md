# Cutlet

A fast and lightweight video trimming app built for speed and simplicity. Cutlet does one thing perfectly: 
trim videos fast.

## Features

- **Lightning Fast** - Hardware accelerated by default. Trim videos instantly with frame-accurate precision.
- **Lightweight** - Minimal resource usage. No bloat, no unnecessary features.
- **Cross-Platform** - Works seamlessly on macOS, Windows, and Linux.

## Download

Download the latest version from the [releases page](https://github.com/brendisurfs/Cutlet/releases).

### macOS
Download the `.dmg` file (Universal binary for Intel and Apple Silicon).

### Windows
Download the `.exe` installer (Windows 10 or later, 64-bit).

### Linux
Download the `.AppImage` file (works on all distributions).

## Installation

### macOS
1. Open the `.dmg` file
2. Drag Cutlet to your Applications folder
3. If you see a security warning, go to System Preferences â†’ Security & Privacy and allow the app

### Windows
1. Run the `.exe` installer
2. Follow the prompts
3. If Windows Defender SmartScreen shows a warning, click "More info" and then "Run anyway"

### Linux
1. Download the `.AppImage` file
2. Make it executable: `chmod +x cutlet_*.AppImage`
3. Run it - no installation required
